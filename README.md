# ☦ Ιερά Μονή Αγίου Κοσμά του Αιτωλού

A responsive static website for the Holy Monastery of Saint Kosmas the Aetolian, built with vanilla HTML, CSS, and JavaScript.

The website presents information about the monastery, its history, publications, and visitor information through a traditional Orthodox-inspired design.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Traditional Greek Orthodox visual style
- Multi-page static website architecture
- Monastery information and history pages
- Publications section with book listings
- Contact page with embedded Google Maps
- Mobile-friendly navigation
- Optimized typography and layout

## 🏛 Pages

### Home

The main landing page introducing the monastery and providing quick access to important sections.

### The Monastery

Contains information about the history, tradition, and spiritual significance of the monastery.

### Publications

Displays the books and publications of the monastery with images and descriptions.

### Contact

Provides visitor information, contact details, and location through Google Maps.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Maps Embed API
- Vercel (Deployment)

## 🎨 Design

The design was inspired by traditional Greek Orthodox monastery websites, using:

- Byzantine-inspired colors
- Warm parchment backgrounds
- Serif typography
- Gold decorative elements
- Minimal and respectful layouts

The goal was to create a digital space that reflects the atmosphere and identity of a monastery.

## 🚀 Deployment

The website is deployed using Vercel.

To run locally:

1. Clone the repository:

```bash
git clone https://github.com/yourusername/your-repository.git
```

2. Open index.html in your browser.

(No build tools or dependencies are required.)
